# GameUp
GameUp is an online ticketing & event-promotion platform for board & card games. GameUp allows you to:
* Create gaming events online
* Buy & sell tickets
* Manage tournament brackets

### Technology Stack
* Parse.com (NoSQL database)
* AngularJS (front-end)
* Stripe (online payment)
* Javascript
